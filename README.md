BibliotecaManager

Sistema de gerenciamento de biblioteca desenvolvido em Java com interface gráfica Swing.

Tecnologias
Java JDK 21
Swing (interface gráfica)
Serialização de objetos (persistência)
Leitura de arquivos TXT
Estrutura
```
BibliotecaManager/
src/
   Main.java
   Pessoa.java
   Aluno.java
   Professor.java
   Emprestavel.java
   Livro.java
   Biblioteca.java
   LivroIndisponivelException.java
   TelaPrincipal.java
dados/
     livros.txt

Como executar
bash
cd src
javac *.java
java Main
